════════════════════════════════════════════════════════════════
  🎯 使用方法
════════════════════════════════════════════════════════════════

【启动开发服务器】
  cd /root/promptvalar/frontend
  npm install    # 首次运行需要安装依赖
  npm run dev    # 启动开发服务器

【访问应用】
  浏览器打开: http://localhost:5173

【测试主题切换】
  1. 找到页面右上角的主题切换按钮（太阳☀️/月亮🌙图标）
  2. 点击按钮切换主题
  3. 刷新页面验证主题保持

【测试响应式】
  1. 按F12打开开发者工具
  2. 点击设备工具栏图标（Ctrl+Shift+M）
  3. 选择不同设备尺寸测试
